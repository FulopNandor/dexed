# Dexed key bindings

| Key bindings                 | Actions                      |
| ---------------------------- | -----------------------------|
| CTRL+1 to CTRL+6             | Focus on operator 'X' group  |
| CTRL+SHIFT+1 TO CTRL+SHIFT+6 | Toggle active operator 'X'   |
| CTRL+Q                       | Focus on global group        |
| CTRL+L                       | Open cartridge manager       |
